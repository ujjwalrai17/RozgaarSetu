## ЁЯЪА Getting Started

### Prerequisites

- Java 11 or higher
- Maven or Gradle
- Spring Boot 2.x or higher
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ujjwalrai17/RozgaarSetu.git
cd RozgaarSetu
```

2. Build the project:
```bash
mvn clean install
```

3. Run the application:
```bash
mvn spring-boot:run
```

The application will start on `http://localhost:8080`
